# Field notes

A plain Markdown file — **not** a notebook. Double-click it in the workspace tree and it
opens in its own **tab**, edited with the same CodeMirror editor the notebooks use.
`Ctrl/Cmd+S` saves it to disk like any file.

This is the "land around the notebooks": a workspace can hold data, docs, configs, and
scripts right next to the `.jl` notebooks that use them.
